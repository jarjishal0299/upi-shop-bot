# UPI Shop Bot

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/jarjishal0299/upi-shop-bot.git
    ```
2. Navigate to the project directory:
    ```bash
    cd upi-shop-bot
    ```
3. Install dependencies (if applicable):
    ```bash
    npm install
    # or use yarn
    yarn install
    ```

## Usage
1. Start the bot:
    ```bash
    node bot.js
    ```
2. Follow the prompts to interact with the bot.
